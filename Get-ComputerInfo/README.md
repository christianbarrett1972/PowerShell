# Description
Retreive info of Computer save on a json file.

**Version: 1.0 - Date: 09/04/2017**
# Parameters
**OutputPath**

Path of json file if not specified current path will used, the name of json file will be 'Hostname.yyyyMMddHHmmss.json'.
